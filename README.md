# IT-Park: HTML va CSS Asoslari - 3-Loyiha (Card Design)

Ushbu loyiha **IT-Park** kursidagi uchinchi amaliy vazifamdir. Bu safar asosiy e'tibor elementlarning tashqi ko'rinishini zamonaviy **Profile Card** ko'rinishiga keltirishga qaratildi. Loyihada elementlarni joylashtirish va ranglar bilan ishlash bo'yicha bilimlar qo'llanildi.

## 🛠 Ishlatilgan texnologiyalar
* **HTML5** — Semantik struktura uchun.
* **CSS3** — Murakkab vizual effektlar va joylashuv (layout) uchun.

## 📝 Loyiha xususiyatlari (CSS Fokus)
Ushbu dizaynni yaratishda quyidagi CSS xususiyatlaridan foydalanildi:

* **Box Model:** Elementlar orasidagi masofani boshqarish uchun `padding` va `margin`.
* **Visual Styling:** Fon ranglari (`background-color`) va matn ranglari (`color`).
* **Border Radius:** Profil rasmini aylana shakliga keltirish uchun `50%` qiymatli chegaralar.
* **Button Design:** Maxsus tugma dizayni va uning interaktivligi.
* **Typography:** Shriftlar o'lchami va qalinligini (`font-weight`) to'g'ri tanlash.

# IT-Park: HTML and CSS Fundamentals - Project #3 (Card Design)

This project is my third practical task at **IT-Park**. The main focus this time was on styling elements to create a modern **Profile Card** UI. I applied my knowledge of element positioning and color schemes to achieve this professional look.

## 🛠 Technologies Used
* **HTML5** — For semantic structure.
* **CSS3** — For advanced visual effects and layout.

## 📝 Project Features (CSS Focus)
The following CSS properties were used to create this design:

* **Box Model:** Using `padding` and `margin` to control spacing between elements.
* **Visual Styling:** Applying `background-color` and text `color` schemes.
* **Border Radius:** Creating a circular profile image using `border-radius: 50%`.
* **Button Design:** Custom styling for the "View profile" button.
* **Typography:** Adjusting font sizes and `font-weight` for better hierarchy.
